# ReadMore - Books

## Beschrijving
ReadMore Books is een WPF-desktopapplicatie voor het beheren en bestellen van boeken. De applicatie biedt functionaliteiten voor zowel gewone gebruikers als administrators:

- **Gebruikers**: boeken zoeken, admins contacteren en bestellingen plaatsen.
- **Admins**: boeken toevoegen, bewerken en verwijderen, bestellingen beheren, gebruikers beheren en contactberichten afhandelen.

De applicatie gebruikt **Entity Framework Core** voor database-interactie en **ASP.NET Identity** voor gebruikersbeheer.

## Installatie & Gebruik
1. Clone de repository.
2. Configureer de connection string in `appsettings.json` voor je database.
3. Bouw het project in Visual Studio.
4. Start de applicatie via `MainWindow`.
